# API

## Introduction

We are using [Hono](https://hono.dev/) to build the API.
We have a Next.js 15 app that is using the API.

We need to programatically make endpoints for a basic CRUD todo app.

Make sure to organize the files well in the API app.

And then use tRPC to connect the API to the Next.js app.

## Endpoints

### Get Users

```ts
// GET /api/users
```

### Get User by ID

```ts
// GET /api/users/:id
```

### Get Todos

```ts
// GET /api/todos
```

### Get Todo by ID

```ts
// GET /api/todos/:id
```

### Create Todo

```ts
// POST /api/todos
```

### Update Todo

```ts
// PUT /api/todos/:id
```

### Delete Todo

```ts
// DELETE /api/todos/:id
```
