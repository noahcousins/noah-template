# Noah Template

Welcome to **Noah Template** - a modern, robust full-stack application template built with Next.js, Hono, and DrizzleORM. This monorepo provides a complete foundation for building scalable web applications with authentication, database integration, and deployment infrastructure.

<Note title="Project Status">
  This project is actively maintained and developed. While suitable for production use,
  always review security practices and dependencies before deploying to production.
</Note>

## Project Overview

Noah Template is a comprehensive starter kit that combines the best modern web technologies into a cohesive development experience. It's designed to accelerate development while maintaining flexibility and scalability.

### Key Features

| **Feature**               | **Description**                                                           |
| ------------------------- | ------------------------------------------------------------------------- |
| Monorepo Structure        | Organized with Turborepo for efficient package management                 |
| Next.js Web Frontend      | Modern React framework with server components and optimized rendering      |
| Hono API Backend          | Lightweight, high-performance API framework running on Cloudflare Workers |
| DrizzleORM Database       | Type-safe ORM with PostgreSQL integration and migration tools             |
| Better Auth               | Complete authentication system with email and social login                 |
| Shadcn UI Components      | Beautiful, accessible UI components with Tailwind CSS                      |
| Environment Management    | Structured environment variables with validation                           |
| Documentation             | Comprehensive documentation site built with MDX                            |

## Technology Stack

The template leverages a modern technology stack:

- **Frontend**: Next.js 14, React, Tailwind CSS, Shadcn UI
- **API**: Hono, Cloudflare Workers
- **Database**: PostgreSQL, DrizzleORM, Neon Database
- **Authentication**: Better Auth with multiple providers
- **Infrastructure**: Turborepo, pnpm workspaces
- **Development**: TypeScript, ESLint, Prettier

## Architecture

Noah Template follows a microservice-inspired architecture with clear separation of concerns:

```
noah-template/
├── apps/
│   ├── web/        # Next.js web application
│   ├── api/        # Hono API service
│   └── docs/       # Documentation site
└── packages/
    ├── db/         # Database schema and utilities
    ├── ui/         # Shared UI components
    ├── env/        # Environment variables management
    └── eslint-config/ # Shared ESLint configuration
```
